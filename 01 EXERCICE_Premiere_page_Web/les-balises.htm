LES BALISES HTML

NOM FONCTIONS PROPRIETES FERMER LA BALISE
BR Revenir à la ligne NON
P Encadre un paragraphe ALIGN OUI
FONT Formats de caractères COLOR SIZE FACE OUI
B Gras OUI
I Italique OUI
U Souligné OUI
BODY Couleur de fond BGCOLOR
BODY Image de fond BACKGROUND
HR Ligne séparatrice COLOR SIZE WIDTH ALIGN NON
IMG Insérer une image ALIGN WIDTH NON
A Créer un lien hypertexte HREF OUI
A Créer un lien interne HREF = "#ANCRE" a name="ANCRE" OUI
OL Créer une liste ordonnée TYPE OUI
UL Créer une liste non ordonnée TYPE OUI
Hx Créer un titre avec une taille ALIGN OUI
BLOCKQUOTE Créer un retrait OUI
&NBSP Créer un espace NON

SYNTAXE D'UNE BALISE

<NOM [propriétés]> Page web </NOM>
propriétés : P1=V1 P2=V2 P3=V3 ...

Exemple :
<FONT Color="red" face="arial" size="5"> Page web </FONT>
Résultat : Page web
